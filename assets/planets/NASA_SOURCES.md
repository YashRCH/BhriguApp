# NASA/JPL Planet Asset Sources

These app-sized JPG assets were resized from NASA/JPL imagery for UI icons.
NASA media usage guidance: https://www.nasa.gov/nasa-brand-center/images-and-media/

- `planet_sun.jpg`: PIA03149, ESA/NASA/SOHO, https://photojournal.jpl.nasa.gov/catalog/PIA03149
- `planet_moon.jpg`: compatibility copy of the existing app asset `moon_phase_waxing_gibbous.png`
- `planet_mercury.jpg`: PIA00437, NASA/JPL/USGS, https://www.jpl.nasa.gov/images/pia00437-planet-mercury
- `planet_venus.jpg`: NASA SVS 30357 / PIA00104, NASA/JPL, https://svs.gsfc.nasa.gov/30357
- `planet_earth.jpg`: PIA00123, NASA/JPL, https://photojournal.jpl.nasa.gov/catalog/PIA00123
- `planet_mars.jpg`: PIA00407, NASA/JPL/USGS, https://photojournal.jpl.nasa.gov/catalog/PIA00407
- `planet_jupiter.jpg`: PIA02873, NASA/JPL/University of Arizona, https://photojournal.jpl.nasa.gov/catalog/PIA02873
- `planet_saturn.jpg`: PIA00400, NASA/JPL/USGS, https://photojournal.jpl.nasa.gov/catalog/PIA00400
- `planet_uranus.jpg`: PIA01391, NASA/JPL, https://photojournal.jpl.nasa.gov/catalog/PIA01391
- `planet_neptune.jpg`: PIA01492, NASA/JPL, https://photojournal.jpl.nasa.gov/catalog/PIA01492
- `planet_pluto.jpg`: PIA20291, NASA/Johns Hopkins University Applied Physics Laboratory/Southwest Research Institute, https://photojournal.jpl.nasa.gov/catalog/PIA20291
